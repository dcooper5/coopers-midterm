Below you will find three different layouts for your webpages.

Liquid Layout

A liquid layout means that all the containers on the page have their dimensions defined in percents. They are based on viewpoint, so the size of your browser window effects their size.

Fixed Width Layout

A fixed layout is exactly what it sounds like, it is a fixed page meaning no matter the viewpoint it will look the same for every page visit. (mobile or desktop)

Grid-Based Layout

A grid layout is helpful when you need to divide certain parts of the page. These work like tables because it allows you to align elements into rows and columns.
